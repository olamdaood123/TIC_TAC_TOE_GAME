# TIC_TAC_TOE_GAME
إضافة جديدة لمشاريعي البرمجية، وهي لعبة "إكس-أو" التقليدية مطورة باستخدام لغة C#. يركز هذا المشروع على تطبيق منطق برمجي متقدم وتوظيف أدوات واجهة المستخدم بشكل فعال لتقديم تجربة لعب سلسة ومنظمة.
المميزات التقنية في اللعبة
منطق التحكيم الذكي: خوارزمية مخصصة لفحص الفوز في جميع الاتجاهات (أفقي، عمودي، وقطري).
كشف التعادل: نظام مراقبة لعدد الضغطات يعلن التعادل في حال امتلاء اللوحة دون فائز.
استخدام الحلقات التكرارية (Loops): تم استخدام foreach لتنظيف اللوحة وإعادة تشغيل اللعبة أو قفل المربعات، مما يضمن كوداً برمجياً نظيفاً وفعالاً.
إدارة الأدوار: نظام تبديل تلقائي بين اللاعب الأول والثاني مع حماية المربعات المشغولة من التغيير.
المفاهيم البرمجية المطبقة
الدوال (Methods): تقسيم الكود إلى وظائف مستقلة (مثل فحص الفوز، قلب الدور، وقفل اللوحة).
المجموعات والعناصر (Controls Collection): التعامل مع أدوات الواجهة (PictureBox) كمجموعة واحدة والتحكم في خصائصها برمجياً عبر الـ Loop.
الأحداث (Events): استخدام حدث ضغط موحد لجميع المربعات لتقليل تكرار الكود.
الشروط المنطقية (Conditional Logic): بناء شروط دقيقة للتحقق من حالة اللعبة بعد كل حركة.
آلية العمل
تبدأ اللعبة باللاعب الأول (X).
عند فوز أي لاعب، يتم قفل اللوحة فوراً وتظهر رسالة بنتيجة المباراة.
في حال التعادل، يتم تنبيه اللاعبين وتوقف اللعبة.
زر RESTART يقوم بإعادة ضبط كل المتغيرات والمربعات بضغطة واحدة للبدء من جديد.
تم تطوير هذا المشروع وتوثيقه ضمن رحلة تطوير مهاراتي في لغة C# - المستوى الأول.
